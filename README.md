# Inventory Management Dashboard

Modern inventory and stock management dashboard built with HTML, CSS and JavaScript.

---

## Features

✅ Add products dynamically  
✅ Delete products  
✅ Real-time inventory statistics  
✅ Low stock alerts  
✅ LocalStorage persistence  
✅ Modern futuristic UI  
✅ Responsive dashboard layout  

---

## Technologies

- HTML5
- CSS3
- JavaScript
- LocalStorage
